# nutrition-assessment(About Project)

```
Description: This application all about getting the Nutrition Data details of favourite food items of your choice,you can get this information by clicking on the favourite fooditems displayed and by searching the food. 

This application implemented using the Edamam API from https://api.edamam.com/api/ for getting Nutrition details of of food choice.

```
#### Technologies Used

```
This Application is build using VueJS as front end Js framework and Vuetify Material Ui components  with possible best practices.

Vue Js is a progressive framework with a small size of around ~21kb , easy to understand and scale large application.
```
